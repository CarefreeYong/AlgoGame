#### AlgoGame
一个基于 UniApp 框架开发的算法可视化与应用的项目。

#### 技术栈
UniApp、Vue3、TypeScript、TSX、Vite

#### 兼容性
项目仅在微信小程序平台中开发与调试，其他平台不保证能正常运行

#### 安装依赖
推荐
```bash
pnpm install
```
或
```bash
npm install
```

#### 构建
推荐
```bash
pnpm dev:mp-weixin
```
或
```bash
npm dev:mp-weixin
```

#### 运行/调试
将 dist/dev/mp-weixin 文件夹导入到 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/stable.html)
